Quick Start
Up & Running in Minutes

Requires Docker Desktop, Node.js 20+, and pnpm. Everything else is automated.

1
Clone the repository
git clone https://github.com/Alirewa/cyber-toolkit.git
cd cyber-toolkit
2
Copy environment config
Copy-Item .env.example .env
Defaults work out of the box — no edits needed for local use.
3
Launch everything with one command
.\start-local.ps1
Starts PostgreSQL container, installs deps, runs migrations, starts API + Web.
4
Open the dashboard
Navigate to http://localhost:4000 — goes straight to your dashboard. No login required.
